This all started with a man named "Ceaser".

This application was created to encrypt the text using Ceaser's method. It can be used to encrypt the message and the user can decrypt it in the same application using the shift number.

If you want to send a message to your friends or relatives secretely send the encrypted message and tell secretly the shift number you have used to encrypt the message so that they can decrypt it later.
